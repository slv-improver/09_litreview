# LITReview

## Instructions to get the app

- `git clone https://github.com/slv-improver/09_litreview.git`
- `cd 09_litreview`
- `python -m venv env`
- `source env/bin/activate`
- `pip install -r requirements.txt`
- `python manage.py runserver`
